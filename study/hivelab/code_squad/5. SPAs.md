# SPAs

왜 쓰는가?
화면 이동시 흰 화면이 나오는게 싫음
새로고침 안되게 하려고
핵심기술은 ajax
사용자의 네비게이터를 막는 단점이 있어 router를 사용할 수 밖에 없다.
(사용자는 뒤로가기 한다.)
라우터를 공부하려면 historyAPI를 공부해봐

모든 프레임워크의 핵심은 model을 view에서 어떻게 분리하느냐 이다
view간에는 컨트롤러로 인해 결합도가 낮아진다. 모든 관계를 어디선가는 알고 있다.
vue, react같은 프레임워크의 장점은 단방향 흐름, 유니다이렉션이다.
ui가 트리구조이기 때문에 가능하다. 데이터가 컴포넌트 단위로 흐르는것(유니다이렉션)
