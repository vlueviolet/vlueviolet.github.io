# ES6 시작하기
* es6 === es2015

## es6의 의미
* 개선된 javascript 문법
* browser compatibility의 훌륭한 지원
    * 해외에서는 거의 표준처럼 사용됨, 국내는 ie 점유율로 제한적 사용
    * babel등 활용하여 es6 -> es5로  변환과정을 거쳐 사용
    * 많은 라이브러리에서 채택함
* es6를 기반으로 한 javascript 생태계의 확산
